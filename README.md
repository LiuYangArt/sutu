# PaintBoard

个人用绘画软件，支持 Wacom 压感输入，基于 Tauri + React + Rust 构建。目前只支持windows平台。 
目前实现了webgpu笔刷，处理大笔刷时性能应该还不错。
目标是做一个接近ps绘画体验的轻量级项目。
